# Heirloom® Rug Cleaning

Area Rug Cleaning in Calgary since 1967 — website built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3011](http://localhost:3011) to view the site.

## Docker

```bash
# Development
./docker.sh up

# Production
./docker.sh up:prod

# Test production build
./docker.sh test
```

## Environment

Copy `.env.example` to `.env.local` and configure the required variables.
